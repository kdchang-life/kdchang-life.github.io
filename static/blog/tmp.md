>「專注於有能力改變的事情，會使我們的力量變得更強大。將所有能量都投注到我們實際上無法影響的事情，是自我放縱和自我毀滅。」

📍 前言

今天為大家介紹這本書是科技和個人成長圈知名的哲學✕行銷鬼才：萊恩．霍利得所寫的《挫折逆轉勝：認知×行動×意志，32個聰明應對困境的心理技巧》。在追求成功的道路上，我們往往將挫折視為不幸的阻礙，卻忽略了應對困境的態度才是決定勝負的關鍵。本書作者萊恩．霍利得融合古典斯多葛哲學與現代商業實務，提出一套系統化的逆境學。他認為，障礙並非前進的終點，而是鍛鍊自我、實現翻轉的絕佳機會。透過修煉認知、行動與意志，我們能將生命的絆腳石轉化為向上的跳板。

📍 重點摘要

1. 認知的修練：學習客觀看待事物，將情緒與事實分離。我們無法控制發生的意外，但能控制對事件的觀點，並從中尋找隱藏的良機。

2. 行動的修練：與其等待完美時機，不如主動出擊。運用創造力與柔性策略，在障礙面前不硬碰硬，而是尋找切入點，讓阻力轉化為前進的動力。

3. 意志的修練：建立內在堡壘以因應不可控的失敗。透過負面觀想預作準備，接受命運的安排，並在百折不撓中尋求比個人私利更廣大的意義。

4. 轉化負向情境：真正的強大不在於無視痛苦，而在於深信「我可以讓這一切變得更好」，將困境視為測試能力的實驗場。


📍 應用行動方案

1. 練習情緒脫鉤：當遇到不如意時，先停頓三秒，問自己「這件事的客觀事實是什麼？」而非陷入「這對我不公平」的情緒泥淖。

2. 拆解目標：面對龐大障礙，採取按部就班的策略。將任務拆解至最小行動單位，專注於目前能做好的每一件小事。

3. 執行負面觀想：定期預演最壞的情況，並思考對策。這能降低對未來的恐懼，增強心理韌性。

4. 主動尋找替代方案：當正面進攻無效時，嘗試側著走或往後退，尋找非傳統的解決路徑。

📍# 問題討論

1. 當我們抱怨外在環境如經濟或小人時，有多少比例是因為我們放棄了內在認知的選擇權？

2. 在追求完美的過程中，我們是否因為害怕不夠完美而停止了有效的行動？

3. 如何在面對無法改變的巨大創傷時，依然能實踐斯多葛學派所說的「熱愛命運」？

4. 意志的修練如何幫助我們在長期的失敗中，保持重啟的勇氣而非落入習得性無助？

📍# 總結

挫折逆轉勝的核心在於思維框架的重構。這本書告訴我們，障礙不是路上的阻礙，障礙本身就是道路。當我們學會不再逃避困境，而是將其視為助長志向的燃料，人生的高度將從此改寫。每一場危機都是優秀者的洗禮，只要掌握認知、行動與意志的循環，我們便能從不幸中奪回幸運的主動權。
